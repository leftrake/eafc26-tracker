# EAFC 26 Career Mode Tracker

A full career mode management suite built in vanilla HTML/JS.

## Files
- `eafc26_tracker_v12.html` — Squad tracker, player profiles, transfers
- `eafc26_matches.html` — Match log with lineup builder
- `eafc26_calendar.html` — In-game calendar and scheduling
- `eafc26_history.html` — Club history timeline
- `eafc26_report.html` — AI-ready squad report and PDF export

## Usage
Open any file locally in your browser. All data is stored in localStorage
and shared across all pages automatically.
